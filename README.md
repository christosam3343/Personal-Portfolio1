## Run locally

Open `index.html` in a browser.

## Easiest free hosting choices

1. GitHub Pages: best if you want a clean developer-friendly workflow and a free `yourusername.github.io` domain.
2. Cloudflare Pages: great performance and gives you a free `your-site.pages.dev` domain.
3. Netlify: simple drag-and-drop or Git deploys with a free `your-site.netlify.app` domain.
4. Vercel: smooth Git-based deploys with a free `your-project.vercel.app` domain.

## Free custom-style domain options

- `yourname.is-a.dev` via is-a.dev
- A paid custom domain like `yourname.com` is still the most professional long-term option, but the platforms above all give you free subdomains if you want to launch now at no cost.
